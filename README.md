# README

## 概要

このプロジェクトは Next.js を使用して構築されたアプリケーションです。

## 環境構築手順

リポジトリをクローンします

```bash
git clone https://github.com/Souax/Task.git
```

プロジェクトルートに移動します

```bash
cd next.js-task
```

パッケージをインストールします

```bash
npm install
```

ローカル環境の立ち上げです

```bash
npm run dev:port
```

下記のローカル環境にアクセスできれば大丈夫です

http://localhost:3001/

## 開発環境

- OS：Linux(CentOS)
- 言語：HTML,CSS,JavaScript,TypeScript
- フレームワーク：Next.js
- JS ライブラリ：React
- IDE：Visual Studio Code
